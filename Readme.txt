Group 3 - Readme

1. Start terminal in the 'FlaskApp' Folder 

2. Run the flask_api.py using terminal (Install numpy, tensorflow, or other requirements if asked)

	- python flask_api.py

3. If the server is hosted, You can start the unity game

4. Click Play 

5. Choose the desired model, and Click Play

6. Use the keys on right to navigate around the map

7. Click on regenerate button if you want to regenerate the map.
